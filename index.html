Simple Support Page

This is a simple implementation of a support page for a service. It offers the ability to submit anonymous bug reports and provide feedback along with an admin portal to view all submit reports. Data storage is done via Parse SDK.

Features

  • Anonymous bug reporting
  • Screenshot support
  • Feedback page
  • Automatic report numbering
  • Admin web portal
    • Assign issues to team members
    • Mark issues as open, fixed, or rejected
    • Comment on individual reports

Requirements

  • Bootstrap v3.3.1 or greater
  • Bootstrap Select
  • Parse SDK

Demo

Public Page

This is the general page that will be accessed by the public to report on bugs and submit feedback as well. For the purposes of this demo, submitted reports and feedback will be purged on a daily basis.
